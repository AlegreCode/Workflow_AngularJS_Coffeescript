Configurar host virtual

IMPORTANTE: este proyecto fue testeado en un host virtual con xampp en un entorno Windows. En esta sección te explicaré como configurarlo, para que puedas usarlo sin complicaciones.

Configurar Host

El archivo hosts de windows, sirve para asignar nombres de host a direcciones ip. La ubicación del archivo se encuentra en: C:\Windows\System32\drivers\etc\host. Dentro de este directorio vas a encontrar un archivo llamado hosts. Abrí el bloc de notas como administrador para que te permita hacer cambios.

En caso de que el directorio aparezca vacío, cambiar la opción del combox "documentos de texto (.txt)" por "todos los archivos (*.*)".

El archivo lucirá algo similar a esto:

La parte resaltada es en dónde tienes que definir tu host. 127.0.0.1 es la dirección ip de la máquina local. Luego pones el nombre del host, este puede llamarse como quieras, en mi caso para este proyecto yo lo llamé angularjscoffeescript.com.

Configurar Host Virtual en XAMPP

Para configurar el host en xampp, tenés que editar el archivo httpd-vhosts.conf. Este archivo se encuentra en la siguiente ruta: C:\xampp\apache\conf\extra. Al igual que en el caso anterior, verifica que el combox muestre todos los archivos.

Encontrarás algo similar a esto.

La parte resaltada muestra lo que tenes que agregar; el virtualhost en mi caso es angularjscoffeescript.com en el puerto 80. En DocumentRoot y Directory va la ruta completa de la carpeta fuente src/ del proyecto, en mi caso es: C:\xampp\htdocs\Proyectos\Workflow_AngularJS_Coffeescript\src. Finalmente en ServerName agregas el nombre de tu host, como se muestra en la imagen. La opción ServerAlias no es obligatorio, como podés ver en el ejemplo, la línea está comentada (con # al inicio de la línea). Esto es todo, ahora reinicias tu xampp, y listo.
